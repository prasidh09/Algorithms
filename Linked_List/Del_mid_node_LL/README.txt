Objective:
==========
	To delete a node in the LL, given access only to that node. 

Logic:
======
	The logic is just to copy the contents of the nodes from the node that has to be deleted and just shifting it one step behind. 

Program:
========
	The function del_mid gets the access to the node that has to be deleted. The loop copies the data in the adjacent nodes to the previous nodes till the end.
